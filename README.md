
# 保険比較推奨アドバイザー (React版)

自動車保険3社（損保ジャパン・東京海上日動・三井住友海上）を比較推奨するデモアプリケーション。

## 🚀 起動方法

```bash
# 依存関係のインストール（初回のみ）
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 にアクセス

## 📁 ファイル構成

```
demosample/
├── index.html           # エントリーポイント
├── package.json         # 依存関係
├── vite.config.js       # Vite設定
└── src/
    ├── main.jsx         # Reactエントリー
    └── InsuranceAdvisor.jsx  # メインコンポーネント
```

## ✨ 機能

- 💬 会話ビュー（意向把握シミュレーション）
- 📋 ヒアリングシート（30項目の進捗管理）
- 📊 3社比較ダッシュボード
- 🏆 推奨理由 & トークスクリプト

## 🔧 技術スタック

- React 18
- Vite 5
- Pure CSS（フレームワーク不使用）
